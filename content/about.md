# About This Project

This is a server-side generation (SSG) project powered by AI.

## What We Do

We generate dynamic web pages using AI from markdown content.

## Features

- **Real-time generation**: Pages are generated on-demand
- **AI-powered**: Uses Google's Gemini Flash 2.5
- **Markdown-based**: Write content in simple markdown
- **Streaming support**: Content streams to the browser in real-time

## Technology Stack

- Deno runtime
- Vercel AI SDK
- Google Gemini AI
- TypeScript
